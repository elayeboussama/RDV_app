# RDV App

A native **Android** application for managing appointments (*rendez-vous*), built in Java with Gradle.

## Tech stack

- Java
- Android SDK, Gradle

## Getting started

### Prerequisites

- Android Studio (with the Android SDK)

### Build & run

Open the project in **Android Studio** and run it on an emulator or device, or from the command line:

```bash
cd "native java app"
./gradlew assembleDebug      # build a debug APK
./gradlew installDebug       # install on a connected device/emulator
```

## Project structure

```
native java app/app/    Android application module (source, resources)
native java app/        Gradle build files
```
